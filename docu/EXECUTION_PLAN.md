# LMS 수행 계획서 (초보자용)

> 대상: 처음 프로젝트를 진행하는 학생
> 
> 기준: 지금까지 구현된 LMS 기능 상태

---

## 1) 목표 한 줄 요약
지금 만든 LMS를 **안정적으로 완성(MVP)** 하고, 발표/시연 가능한 수준으로 정리한다.

---

## 2) 프로젝트 주제 + 사용/예정 기술

### 프로젝트 주제
이 LMS는 단순 강의 목록 사이트가 아니라,
**IT 학습을 안내하고 학습 흐름을 관리해주는 학습 안내형 플랫폼**을 목표로 한다.

핵심 방향:
- 학습자가 강의를 찾고(검색/카테고리)
- 수강하고(수강신청/학습하기)
- 진행을 확인하고(진도/성적)
- 피드백을 남기며(수강평/공지 댓글)
- 꾸준히 학습할 수 있게(스케줄/알림 기반) 돕는다.

### 사용 기술
- **Backend:** Java, Spring Boot, Spring Security
- **DB 접근:** JdbcTemplate (직접 SQL 관리)
- **Database:** MySQL
- **View:** Thymeleaf + HTML/CSS/JavaScript
- **Build:** Gradle

### 추가/예정 기술(확장)
- 시험 기능 고도화용 로직/도메인 분리
- 학습노트 CRUD
- 알림(문자/푸시) 연동
- 운영 편의를 위한 관리자 대시보드 확장

---

## 3) 초보자용 실행 순서 (중요)

## Step 1. "동작 확인"부터 한다
- 로그인(일반/사업자) 각각 확인
- 수강신청 -> 내역 반영 확인
- 수강포기 -> 내역/수강생수 반영 확인
- 공지 댓글 작성/수정/삭제 확인
- 수강 스케줄 저장/조회/삭제 확인

✅ 이 단계 목적: "지금 뭐가 되는지"를 정확히 아는 것

---

## Step 2. "버그 목록"을 만든다
아래 템플릿으로 `BUGLIST.md` 만들기:

- 번호
- 재현 방법
- 기대 결과
- 실제 결과
- 우선순위 (상/중/하)

✅ 이 단계 목적: 감으로 수정하지 않고, 재현 가능한 작업으로 바꾸기

---

## Step 3. 우선순위대로 고친다
우선순위 기준:
1. 로그인/권한 오류 (가장 먼저)
2. 데이터 저장/삭제 오류
3. 화면 깨짐
4. 문구/디자인

✅ 팁: 한 번에 크게 고치지 말고, 작은 단위로 고치고 바로 확인

---

## Step 4. 문서 정리
최소 문서 4개 유지:
- `REQUIREMENTS.md` (요구사항)
- `ARCHITECTURE.md` (구조)
- `ERD.md` (DB 관계)
- `EXECUTION_PLAN.md` (이 문서)

✅ 발표/과제에서 "무엇을 만들었는지" 설명력이 급상승함

---

## 4) 2주 수행 일정 (초보자용)

### 1주차
- Day 1: 전체 기능 테스트 + 버그리스트 작성
- Day 2: 로그인/권한/데이터 저장 버그 수정
- Day 3: 수강신청~학습하기 흐름 안정화
- Day 4: 공지/댓글 흐름 안정화
- Day 5: 수강스케줄, 사업자 흐름 점검 + 정리

### 2주차
- Day 1: UI 통일(버튼/간격/폰트)
- Day 2: 예외처리 문구 통일(알림/오류)
- Day 3: 시연 시나리오 작성
- Day 4: 문서 최종 정리
- Day 5: 최종 리허설 + 제출

---

## 5) 시연(데모) 추천 시나리오

1. 일반회원 로그인
2. 강의 검색 -> 수강신청
3. 수강신청내역 -> 학습하기 진입
4. 수강평 작성(진도율 조건 설명)
5. 수강포기
6. 공지사항 댓글 작성/수정/삭제
7. 사업자 로그인 -> 강의관리 메뉴 확인 -> 수강등록 화면 확인

✅ 시연은 "긴 설명"보다 "짧은 성공 흐름"이 중요

---

## 6) 초보자가 자주 하는 실수 & 방지법

- 실수: 코드 먼저 건드림
  - 방지: 재현 -> 기록 -> 수정 순서 고정

- 실수: 여러 기능 동시 수정
  - 방지: 한 파일/한 기능씩 커밋

- 실수: 에러 로그 안 보고 감으로 수정
  - 방지: 로그 첫 에러 원인부터 확인

- 실수: 데이터 초기화 설정 확인 안 함
  - 방지: 실행 설정(`sql.init.mode`) 먼저 체크

---

## 7) 완료 기준 (체크리스트)

- [ ] 일반 로그인/사업자 로그인 정상

- [ ] 수강신청내역/학습하기 흐름 정상

- [ ] 공지 댓글 권한(본인만 수정/삭제) 정상
- [ ] 수강 스케줄 저장/조회/전체삭제 정상
- [ ] UI 깨짐 없음(주요 페이지)
- [ ] 문서 4종 최신 상태
